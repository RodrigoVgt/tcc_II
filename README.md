# tcc_II
Repositório para TCCII

Para todos os conjuntos, é necessário ter o banco de dados a ser utilizado configurado. Para mais detalhes de cada, consultar readme de cada

/scraper:
     contém os arquivos necessários para rodar o scraper. É válido mencionar que alterações no layout do site podem alterar o resultado ou quebrar totalmente a execução do scraper.

/data_treatment&ZeroShot:
     contém os arquivos necessários para rodar o tratamento de dados(gerar falsos e filtrar verdadeiros) e a execução do ZeroShot.

/makeEmbeddings:
     contém os arquivos necessários para gerar as embbedings de perguntas da base de avaliação, bem como dos manuais

/rag_questions:
     contém os arquivos necessários para rodar o RAG com perguntas.

/rag_manuals:
     contém os arquivos necessários para rodar o RAG com manuais.
